# Python 编程课程（人工智能/机器人工程 专业课）

欢迎来到 **Python 编程课程组织** 👋  
本组织用于托管《Python 编程》课程的教学资源与学生作业。  

## 📚 课程简介
- **对象**：人工智能/机器人工程专业 **本科二年级上学期** 学生  
- **安排**：
  - 前 12 周：Python 编程基础（语法、数据结构、函数、模块、面向对象等）

## 🛠️ 使用说明
- 学生将通过 **GitHub Classroom** 获得作业仓库。  
- 每份作业的 starter code 已预置在模板仓库中，请在此基础上完成任务并提交。  
- 所有提交均需通过 **Git + GitHub** 完成，作为课程学习的一部分。  

## 📌 注意事项
1. 仓库请保持私有，仅学生本人和教师/助教可访问。  
2. 提交代码需遵循课程给出的命名规范与提交要求。  
3. 建议学生在本地使用 **Anaconda + PyCharm / VS Code** 完成开发。  
4. 单元测试使用 `pytest`，请确保作业能通过提供的测试用例。  

## 🎯 教学目标
通过本课程，学生将能够：
- 熟练使用 Python 进行编程与调试  
- 掌握基础算法与数据结构的实现  
- 使用 GitHub 进行协作与版本控制  
- 为后续机器学习与人工智能课程打下坚实的编程基础
